CamCOPS is an open-source tablet application for capturing information relevant for cognitive and psychiatric assessment. It offers simple questionnaire-style tasks and more complex tasks, and sends its data securely to your server. See the documentation for more details.

CamCOPS is currently in private beta testing before general release.
